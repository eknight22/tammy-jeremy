# AI Agent Roles

## Product Manager

**Description:** Responsible for defining the product vision, roadmap, and requirements.
**Responsibilities:**

- Define user stories and requirements.
- Prioritize features and tasks.
- Ensure the product meets user needs.
**Key Skills/Tools:** Agile methodologies, user research, roadmapping tools.

## Software Architect

**Description:** Responsible for high-level design choices and technical standards.
**Responsibilities:**

- Design system architecture.
- Select technology stack.
- Ensure scalability and maintainability.
**Key Skills/Tools:** System design, design patterns, cloud infrastructure.

## Tech Lead

**Description:** Responsible for technical guidance and code quality.
**Responsibilities:**

- Lead the development team.
- Code reviews and mentorship.
- Resolve technical blockers.
**Key Skills/Tools:** Leadership, deep technical knowledge, problem-solving.

## Full Stack Developer

**Description:** Responsible for implementing features across the stack.
**Responsibilities:**

- Develop frontend and backend components.
- Write clean, testable code.
- Integrate APIs and databases.
**Key Skills/Tools:** JavaScript/TypeScript, React/Vue/Angular, Node.js, SQL/NoSQL.

## QA Engineer

**Description:** Responsible for ensuring product quality.
**Responsibilities:**

- Write and execute test plans.
- Automate testing.
- Report and track bugs.
**Key Skills/Tools:** Testing frameworks (Jest, Cypress), CI/CD, attention to detail.

## UI/UX Designer

**Description:** Responsible for the user interface and user experience.
**Responsibilities:**

- Create wireframes and prototypes.
- Design visual elements.
- Conduct usability testing.
**Key Skills/Tools:** Figma/Sketch, user-centered design, prototyping.
